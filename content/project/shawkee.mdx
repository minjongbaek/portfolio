> Mac OS의 UI를 웹으로 구현한 토이 프로젝트입니다. 평소에 구현해보고 싶은 기능들을 플레이그라운드 형태로 개발했습니다.<br /> [프로젝트 링크](https://shawkee-os.vercel.app/) &nbsp; [GitHub 링크](https://github.com/shaw-kee/shawkee-os)

`React` `TypeScript` `Tailwind CSS`

- 오디오 재생 관련 로직을 커스텀 훅으로 추상화했습니다.
- Canvas API를 활용하여 Snake 미니게임 앱을 구현했습니다.
- iframe을 활용하여 현재 페이지 내에서 블로그 콘텐츠를 바로 볼 수 있도록 Safari 앱을 구현했습니다.
- [앱 컴포넌트를 리사이징 할 때 state 대신 style을 직접 변경하여 렌더링 최적화 작업을 진행했습니다.](https://github.com/shaw-kee/shawkee-os/pull/21)
